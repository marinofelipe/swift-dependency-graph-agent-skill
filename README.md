# Swift Dependency Graph Agent Skill

Expert guidance for modularizing Swift codebases, designing scalable dependency graphs, and maintaining healthy module boundaries and build performance.

## Status

⚠️ **Work in progress. This skill is not ready for production use yet.**

## Who this is for

- Teams planning a modularization effort and needing a clear target dependency graph.
- Developers tackling cycles, unclear boundaries, or slow builds in Swift projects.

## How to use

Use this skill with any AI tool that supports the open agent skills format. Ask the agent to use the skill when you need help designing or reviewing a Swift dependency graph.

Example prompt:

```text
Use the swift dependency graph skill to propose a target module layering and migration plan for this project.
```

## Development

Validate the skill format:

```bash
make validate
```
